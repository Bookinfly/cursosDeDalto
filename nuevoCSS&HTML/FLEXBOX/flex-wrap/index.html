Cuando dejan de entrar, crea una nueva columna/fila

        
            body { 
                margin: 0;
                font-family: sans-serif;
            }
            
            .flexbox {
                display: flex;
                flex-direction: column;
                flex-wrap: wrap;
                /*con wrap al no entrar pueden ir descendiendo en otra fila*/
                height: 100vh;
            }
            
            .flex-item {
                background-color: #800;
                margin: 10px;
                color: #fff;
                font-size: 40px;
                text-align: center;
                width: 200px;
                direction: rtl;/*derecha a izquierda*/
            }
            
            .flex-item:first-child {
                background-color: black;
            }
            
            .flex-item:last-child {
                background-color: #f88;
            }
